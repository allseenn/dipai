**–û—Ç–≤–µ—Ç** (—Ü–µ–ª–µ–≤–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è) ‚Äî —Ç–æ, —á—Ç–æ –Ω–∞–¥–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞—Ç—å. –£—Å–ø–µ–≤–∞–µ–º–æ—Å—Ç—å –∫–∞–∫ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ –†–µ–∑—É–ª—å—Ç–∞—Ç—ã –ï–ì–≠. –û—Ç–≤–µ—Ç—ã –æ–±–æ–∑–Ω–∞—á–∞—é—Ç—Å—è –±—É–∫–≤–æ–π ùë¶.

**–û–±—É—á–∞—é—â–∞—è –≤—ã–±–æ—Ä–∫–∞** - —ç—Ç–æ –ø—Ä–∏–º–µ—Ä—ã, –Ω–∞ –æ—Å–Ω–æ–≤–µ –∫–æ—Ç–æ—Ä—ã—Ö –º—ã –ø–ª–∞–Ω–∏—Ä—É–µ–º —Å—Ç—Ä–æ–∏—Ç—å –æ–±—â—É—é –∑–∞–∫–æ–Ω–æ–º–µ—Ä–Ω–æ—Å—Ç—å. –û–Ω–∞ –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è $X$ –∏ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ $l$ –ø–∞—Ä –æ–±—ä–µ–∫—Ç–æ–≤ $x_{i}$ –∏ –∏–∑–≤–µ—Å—Ç–Ω—ã—Ö –æ—Ç–≤–µ—Ç–æ–≤ $y_{i}$:

$$X = (x^{i}, y_{i})^l_{i=1}.$$

**–ú–æ–¥–µ–ª—å (–∞–ª–≥–æ—Ä–∏—Ç–º) –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è** - —Ñ—É–Ω–∫—Ü–∏—è, –æ—Ç–æ–±—Ä–∞–∂–∞—é—â–∞—è –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –æ–±—ä–µ–∫—Ç–æ–≤ $\mathbb{X}$ –≤ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ –æ—Ç–≤–µ—Ç–æ–≤ $\mathbb{Y}$, –ø–æ–º–æ–≥–∞—é—â–∞—è –Ω–∞–º –¥–µ–ª–∞—Ç—å –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è, –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è $a(x)$. –û–Ω–∞ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ –æ–±—ä–µ–∫—Ç –∏ –≤—ã–¥–∞—ë—Ç –æ—Ç–≤–µ—Ç.

$$a(x) = sign(w_{0} + w_{1}x^{1}+w_{2}x^{2}+...+w_{d}x^{d}).$$

**–§—É–Ω–∫—Ü–∏—è –ø–æ—Ç–µ—Ä—å** (—Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª –æ—à–∏–±–∫–∏ loss function) $Q(a, X)$ - –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø–æ–¥–±–æ—Ä–∞ –ø–æ–¥—Ö–æ–¥—è—â–µ–π –º–æ–¥–µ–ª–∏ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–π. –û–Ω–∞ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ –º–æ–¥–µ–ª—å –∏ –≤—ã–±–æ—Ä–∫—É –∏ —Å–æ–æ–±—â–∞–µ—Ç, –Ω–∞—Å–∫–æ–ª—å–∫–æ —Ö–æ—Ä–æ—à–æ –º–æ–¥–µ–ª—å —Ä–∞–±–æ—Ç–∞–µ—Ç –Ω–∞ —É–∫–∞–∑–∞–Ω–Ω–æ–π –≤—ã–±–æ—Ä–∫–µ. –ó–∞–¥–∞—á–∞ –æ–±—É—á–µ–Ω–∏—è –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ –ø–æ–¥–±–æ—Ä–µ —Ç–∞–∫–æ–π –º–æ–¥–µ–ª–∏, –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–π —Ñ—É–Ω–∫—Ü–∏—è –ø–æ—Ç–µ—Ä—å –º–∏–Ω–∏–º–∞–ª—å–Ω–∞ $Q(a, X)\rightarrow min.$

**–°–µ–º–µ–π—Å—Ç–≤–æ –º–æ–¥–µ–ª–µ–π (–∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤)** $\mathbb{A}$ - —ç—Ç–æ –∞–ª–≥–æ—Ä–∏—Ç–º—ã —Å—Ä–µ–¥–∏ –∫–æ—Ç–æ—Ä—ã—Ö –≤—ã–±–∏—Ä–∞—é—Ç –Ω–∞–∏–ª—É—á—à–∏–π –¥–ª—è –¥–æ—Å—Ç–∏–∂–µ–Ω–∏—è –º–∏–Ω–∏–º—É–º–∞ —Ñ—É–Ω–∫—Ü–∏–∏ –ø–æ—Ç–µ—Ä—å

**–ú–æ–¥–µ–ª–∏ —Å–µ–º–µ–π—Å—Ç–≤–∞ –ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏** ‚Äî —ç—Ç–æ —Ç–∞–∫–∏–µ –º–æ–¥–µ–ª–∏, –∫–æ—Ç–æ—Ä—ã–µ —Å–≤–æ–¥—è—Ç—Å—è –∫ —Å—É–º–º–∏—Ä–æ–≤–∞–Ω–∏—é –∑–Ω–∞—á–µ–Ω–∏–π –ø—Ä–∏–∑–Ω–∞–∫–æ–≤ —Å –Ω–µ–∫–æ—Ç–æ—Ä—ã–º–∏ –≤–µ—Å–∞–º–∏:
